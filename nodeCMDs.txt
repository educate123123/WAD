Create a Node.js Application Which Serves a Static Website

We will build a simple Node.js application that serves:

HTML
CSS
JavaScript
Images

using Node.js and Express.js.

What is a Static Website?

A static website contains fixed files:

HTML
CSS
JavaScript
Images

The server simply sends these files to browser.

Example:

Portfolio website
College website
Landing page
Why Use Node.js to Serve Static Website?
Advantages
Advantage	Explanation
Fast	Uses asynchronous architecture
Lightweight	Easy to setup
Scalable	Can handle many requests
Easy Hosting	Can deploy easily
Single Language	JavaScript for frontend & backend
STEP 1: Install Node.js

Download and install:

Node.js Official Website

STEP 2: Verify Installation

Open terminal or CMD:

node -v

Check npm:

npm -v
STEP 3: Create Project Folder
mkdir StaticWebsite
cd StaticWebsite
STEP 4: Initialize Node Project
npm init -y

This creates:

package.json

Purpose:

Stores project information
Manages dependencies
STEP 5: Install Express.js
npm install express

Why Express?

Simplifies server creation
Easy routing
Static file serving support

    <link rel="stylesheet" href="style.css">

<body>

    </button>

STEP 8: Create CSS File
public/style.css
body{
    text-align:center;
    font-family:Arial;
    background-color:lightblue;
}

h1{
    color:darkblue;
}
STEP 9: Create JavaScript File
public/script.js
function showMessage(){
    alert("Website is working!");
}
STEP 10: Create Server File
server.js
const express = require('express');

const app = express();

const path = require('path');


// Serve Static Files
app.use(express.static(path.join(__dirname, 'public')));


// Server Port
const PORT = 3000;


// Start Server
app.listen(PORT, () => {
    console.log(`Server running on port ${PORT}`);
});
Understanding the Code
1. Import Express
const express = require('express');

Loads Express framework.

2. Create App
const app = express();

Creates Express application.

3. Import Path Module
const path = require('path');

Used for folder/file paths.

4. Serve Static Files
app.use(express.static(path.join(__dirname, 'public')));

Important line.

Purpose:

Makes files inside public accessible in browser.

Without this:

HTML/CSS/JS won’t load.
STEP 11: Run Server
node server.js

Output:

Server running on port 3000
STEP 12: Open Browser

Visit:

http://localhost:3000

Website will open.

How It Works Internally
Browser Request
       ↓
Node.js Server
       ↓
Express Static Middleware
       ↓
Reads Files from Public Folder
       ↓
Sends Response to Browser

Difference Between Static and Dynamic Website
Static Website	Dynamic Website
Fixed content	Data changes dynamically
Faster	More processing
No database required	Database usually required
Easy hosting	Complex backend
